h2. Overview

This README file refers to the RubyCampus relationship management and fundraising software designed to address the needs of higher educational and non-profit organizations for the purposes of student recruitment, enrollment management, alumni development and fundraising.

The purpose of this file is to enumerate the various versions of RubyCampus along with the appropriate versions of related dependencies needed to successfully deploy and operate your installation.

h2. Minimal Requirements

The current version of the RubyCampus can be installed and executed on any Unix based or inspired operating system such as Linux, FreeBSD and Apple OS X or other compatible operating system.
 
We suggest the following environment but you may choose any number of suitable alternatives depending on your organizations specific requirements and resources.

h3. Recommended 

* Linux/BSD OS
* Apache2
* MySQL 5.0
* Phusion Passenger (mod_rails)

h3. Required    

* Ruby 1.8.7 (Phusion Passenger's Ruby Enterprise Edition build if running Passenger)
* Gem 1.2.0
* Rails 2.1.0
